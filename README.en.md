# ⚡ Beatrix

**Precision Dark** — A precision instrument for sports betting.

Beatrix is not just another prediction website. It is an **optimized decision tunnel**, designed for demanding bettors, featuring a "Low-Gravity" interface that conveys constant mathematical calculation.

## 🛠️ Tech Stack (Antigravity Stack)
- **Framework**: Next.js 16 (App Router)
- **Core**: React 19
- **Styling**: Tailwind CSS v4 (CSS-First via `@theme`)
- **Animations & Icons**: Framer Motion, Lucide Animated
- **Aesthetic**: Linear-inspired (Background `#0A0A0A`, Surfaces `#121212`, Accents `#00FF9F`).

## 🚀 Key Features
- **Hybrid Command Palette (`Cmd+K`)**: Complete control center featuring an input field and a quick-action grid, including "Pro" commands (`/edge`, `/true`, `/clv`).
- **UI Precision Kit**: Premium components (e.g., animated `ProbabilityValue`, `NeonCard` with spotlight effects, luminescent confidence bars).
- **Zero-Click Value**: Immediate highlighting of Value Bets without unnecessary navigation.
