# ⚡ Beatrix

**Precision Dark** — Un instrument de précision pour le betting sportif.

Beatrix n'est pas un simple site de pronostics. C'est un **tunnel de décision optimisé**, conçu pour les parieurs exigeants, avec une interface "Low-Gravity" qui suggère le calcul mathématique constant.

## 🛠️ Stack Technique (Antigravity Stack)
- **Framework** : Next.js 16 (App Router)
- **Core** : React 19
- **Style** : Tailwind CSS v4 (CSS-First via `@theme`)
- **Animations & Icônes** : Framer Motion, Lucide Animated
- **Esthétique** : Inspirée de Linear (Fond `#0A0A0A`, Surfaces `#121212`, Accents `#00FF9F`).

## 🚀 Fonctionnalités Clés
- **Command Palette Hybride (`Cmd+K`)** : Centre de contrôle complet avec saisie et grille d'actions rapides, incluant des commandes "Pro" (`/edge`, `/true`, `/clv`).
- **UI Precision Kit** : Composants premium (ex: `ProbabilityValue` animé, `NeonCard` avec effet spotlight, barres de confiance luminescentes).
- **Zero-Click Value** : Mise en évidence immédiate des valeurs (Value Bets) sans navigation superflue.
